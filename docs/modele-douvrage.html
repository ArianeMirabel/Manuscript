6 Modèle d’ouvrage

Ce modèle s’appuie sur Bookdown pour permettre d’écrire en R Markdown un livre, un rapport, une thèse de doctorat ou un mémoire d’HDR.

Le fichier principal est index.Rmd qui contient la description de l’ouvrage dans son entête. Tous les autres fichiers .Rmd du dossier contiennent un chapitre. Le fichier MyBook.bib contient la bibliographie.

Ce fichier devra être supprimé, comme 01-Demarrage.Rmd et 02-Syntaxe.Rmd : ils doivent être remplacés par le contenu de l’ouvrage.

Pour commener, ouvrir le projet R qui se trouve dans ce dossier. Ouvrir ensuite index.Rmd et cliquer sur le bouton Build Book dans la fenêtre Build de Rstudio.
